# BibleRogue2 2–5장 스토리 로드맵

이 문서는 1장 「출애굽」 이후 홈 화면에 표시될 2–5장의 주제, 핵심 장면, 선택 구조, 일러스트 방향을 정리한 기획 문서이다.

현재 홈 화면 기준:

1. 출애굽
2. 여리고
3. 갈멜산
4. 사자굴
5. 빈 무덤

---

## 공통 설계 원칙

### 플레이어 위치

플레이어는 성경의 주인공을 대체하지 않는다.

- 모세가 되지 않는다.
- 여호수아가 되지 않는다.
- 엘리야가 되지 않는다.
- 다니엘이 되지 않는다.
- 막달라 마리아나 제자가 되지 않는다.

플레이어는 사건을 목격하고, 공동체 안에서 반응하는 이름 없는 증인이다.

### 트루엔딩 기준

트루엔딩은 성경의 큰 흐름을 따라간다.
배드엔딩은 성경 본문을 바꾸는 것이 아니라, 플레이어와 주변 공동체가 잘못 반응했을 때 생기는 가상 분기이다.

### 선택지 기준

좋은 선택지는 정답/오답이 너무 노골적이지 않아야 한다.

나쁜 예:

```text
하나님을 믿는다 / 하나님을 배신한다
```

좋은 예:

```text
두려움 속에서도 말씀이 가리키는 방향을 따른다
더 확실한 증거가 있을 때까지 기다리자고 말한다
뒤처진 사람들을 먼저 살핀다
```

# 2장. 여리고

## 본문 범위

여호수아 2장, 5:13–6:27 중심

## 홈 카드 문구

```text
2장 · 여리고
붉은 줄과 무너진 성
```

## 핵심 주제

- 정탐꾼의 두려움과 라합의 믿음
- 성벽 안의 소문과 공포
- 붉은 줄의 표지
- 침묵하며 도는 백성
- 무너지는 성과 살아남는 집

## 플레이어 위치

이스라엘 진영 안의 이름 없는 증인.
여리고 정탐 소식을 듣고, 라합의 집 이야기를 전해 듣고, 성을 도는 공동체 안에서 반응한다.

## 핵심 질문

```text
보이지 않는 약속을 믿고 침묵의 걸음을 걸을 수 있는가?
이방 여인의 믿음을 하나님의 이야기 안에서 받아들일 수 있는가?
```

## 예상 노드 흐름

```text
요단을 건넌 후의 진영
→ 정탐꾼이 돌아온 밤
→ 라합의 붉은 줄 이야기
→ 여호와의 군대 대장 앞의 두려움
→ 첫째 날의 침묵
→ 반복되는 행진
→ 성 안의 조롱과 진영의 흔들림
→ 일곱째 날
→ 외침과 무너지는 성
→ 라합의 집
→ 여리고 이후의 침묵
→ 엔딩
```

## 주요 선택축

- 침묵 / 조급함
- 이방인 수용 / 배제
- 공동체 질서 / 개인 판단
- 약속 기억 / 성벽 공포

## 상태값 활용

- `trust`: 말씀과 약속 신뢰
- `community`: 행진 질서와 공동체 유지
- `discernment`: 라합 사건과 전쟁 상황 분별
- `memory`: 붉은 줄과 약속 기억
- `fear`: 성벽과 전쟁 공포
- `delay`: 순종 지체
- `scatter`: 공동체 흩어짐

## 배드엔딩 후보

### 붉은 줄을 끊은 자

라합의 집을 가볍게 여기거나 약속을 배신하는 선택에서 발생한다.

### 침묵을 깨뜨린 진영

여리고를 도는 명령을 조급함과 불신으로 깨뜨릴 때 발생한다.

### 성벽 앞에 멈춘 사람들

성벽의 위압감 앞에서 전진을 포기할 때 발생한다.

## 홈/플레이 일러스트 방향

```text
고대 가나안 성벽, 새벽빛, 이스라엘 진영의 실루엣, 성벽 창문에 붉은 줄, 긴장된 침묵, 높은 명암 대비
```

# 3장. 갈멜산

## 본문 범위

열왕기상 17–19장 중 18장을 중심으로 구성

## 홈 카드 문구

```text
3장 · 갈멜산
침묵을 깨는 불
```

## 핵심 주제

- 가뭄과 기근
- 우상숭배와 혼합 신앙
- 엘리야와 바알 선지자들
- 응답 없는 외침
- 하늘에서 내려온 불
- 작은 구름과 다시 내리는 비
- 승리 이후의 탈진

## 플레이어 위치

북이스라엘 백성 가운데 있는 이름 없는 증인.
바알과 여호와 사이에서 머뭇거리는 백성의 한 사람으로 사건을 목격한다.

## 핵심 질문

```text
침묵하는 우상과 응답하시는 하나님 사이에서 어디에 설 것인가?
불이 내린 뒤에도 사람은 왜 다시 두려워지는가?
```

## 예상 노드 흐름

```text
마른 땅과 닫힌 하늘
→ 아합의 궁정 소문
→ 갈멜산으로 모이는 백성
→ 둘 사이에서 머뭇거리는 사람들
→ 바알 선지자들의 외침
→ 무너진 제단을 다시 쌓음
→ 물을 붓는 명령
→ 하늘에서 내려온 불
→ 백성의 고백
→ 작은 구름
→ 비의 소리
→ 엔딩
```

## 주요 선택축

- 머뭇거림 / 결단
- 구경꾼 / 증인
- 우상 질서 / 여호와 신앙
- 불의 감격 / 비를 기다리는 인내

## 상태값 활용

- `trust`: 여호와의 응답을 기다림
- `discernment`: 바알 숭배와 정치 권력 분별
- `memory`: 조상들의 하나님 기억
- `fear`: 아합과 이세벨 권력에 대한 두려움
- `community`: 흔들리는 백성 사이의 고백 확산

## 배드엔딩 후보

### 둘 사이에 머문 자

결단하지 못하고 계속 양쪽을 오가는 선택에서 발생한다.

### 불을 보고도 돌아선 자

응답을 목격하고도 권력의 보복이 두려워 침묵할 때 발생한다.

### 메마른 마음

비가 오기 전 기다림을 포기할 때 발생한다.

## 홈/플레이 일러스트 방향

```text
마른 갈멜산, 무너진 제단, 어두운 하늘, 불빛과 연기, 멀리 선 백성의 실루엣, 극적인 대비
```

# 4장. 사자굴

## 본문 범위

다니엘 6장 중심

## 홈 카드 문구

```text
4장 · 사자굴
닫힌 굴과 열린 믿음
```

## 핵심 주제

- 제국의 법과 신앙의 습관
- 다니엘을 향한 모함
- 기도를 금지하는 조서
- 열린 창문 앞의 기도
- 왕의 무력함
- 사자굴의 밤
- 새벽의 확인
- 건져냄과 증언

## 플레이어 위치

바벨론/바사 제국 안에서 살아가는 유다 포로 공동체의 이름 없는 증인.
다니엘의 기도와 체포, 사자굴의 밤을 주변에서 목격한다.

## 핵심 질문

```text
법이 신앙을 금지할 때, 오래된 기도의 습관은 어떻게 증언이 되는가?
살아남는 것이 목적이 아니라 하나님께 속한 삶을 지키는 것이 목적일 수 있는가?
```

## 예상 노드 흐름

```text
제국의 관청
→ 다니엘을 시기하는 사람들
→ 금지 조서가 내려온 날
→ 열린 창문 앞의 기도
→ 체포의 소문
→ 왕의 밤
→ 사자굴의 문
→ 닫힌 돌
→ 새벽의 부름
→ 살아 있는 다니엘
→ 제국 안의 증언
→ 엔딩
```

## 주요 선택축

- 안전 / 신앙의 습관
- 침묵 / 증언
- 제국의 법 / 하나님의 통치
- 두려움 / 기도

## 상태값 활용

- `trust`: 하나님께 맡기는 신뢰
- `memory`: 예루살렘을 향한 기도 기억
- `discernment`: 제국의 법과 모함 분별
- `fear`: 처벌과 감시의 두려움
- `community`: 포로 공동체를 격려함

## 배드엔딩 후보

### 닫힌 창문

위험을 피하려고 기도의 창문을 닫고 침묵을 선택할 때 발생한다.

### 제국의 언어만 남은 자

제국의 법과 성공을 절대화할 때 발생한다.

### 새벽을 기다리지 못한 사람들

사자굴의 밤에 공동체가 흩어질 때 발생한다.

## 홈/플레이 일러스트 방향

```text
고대 제국 궁정, 밤의 사자굴, 봉인된 돌문, 새벽빛, 굴 안의 사자 실루엣, 다니엘의 희미한 형상, 높은 명암 대비
```

# 5장. 빈 무덤

## 본문 범위

마태복음 28장, 마가복음 16장, 누가복음 24장, 요한복음 20장 중 하나를 기준 본문으로 정해야 한다.
초기 구현에서는 요한복음 20:1–18 또는 누가복음 24:1–12를 추천한다.

## 홈 카드 문구

```text
5장 · 빈 무덤
새벽의 목격자들
```

## 핵심 주제

- 안식 후 첫날 새벽
- 돌이 옮겨짐
- 빈 무덤
- 두려움과 놀람
- 천사의 말 또는 예수님의 부르심
- 제자들에게 전하는 증언
- 믿기 어려운 기쁜 소식

## 플레이어 위치

예루살렘 근처의 이름 없는 증인.
여인들의 증언, 제자들의 혼란, 빈 무덤의 소식을 따라 움직인다.

## 핵심 질문

```text
상실의 자리에서 들려온 부활의 소식을 어떻게 받아들일 것인가?
빈 무덤은 공허인가, 새 창조의 시작인가?
```

## 예상 노드 흐름

```text
안식 후 첫날 새벽
→ 무덤으로 가는 길
→ 옮겨진 돌
→ 비어 있는 자리
→ 두려움 속의 질문
→ 이름을 부르는 음성 또는 천사의 증언
→ 제자들에게 달려감
→ 믿지 못하는 사람들
→ 다시 무덤으로 향하는 발걸음
→ 부활의 증언
→ 엔딩
```

## 주요 선택축

- 상실 / 소망
- 두려움 / 증언
- 확인하려는 마음 / 부르심에 응답
- 혼자 간직함 / 공동체에 전함

## 상태값 활용

- `trust`: 부활 소식을 받아들임
- `memory`: 예수님의 말씀을 기억함
- `community`: 제자들에게 전함
- `discernment`: 빈 무덤의 의미를 분별함
- `fear`: 두려움과 침묵
- `delay`: 전하기를 지체함

## 배드엔딩 후보

### 닫힌 새벽

두려움 때문에 아무에게도 말하지 못하고 침묵할 때 발생한다.

### 시체를 찾는 사람

부활의 소식을 듣고도 죽음의 방식으로만 사건을 해석할 때 발생한다.

### 전해지지 않은 소식

공동체로 달려가지 않고 자기 안에만 갇힐 때 발생한다.

## 홈/플레이 일러스트 방향

```text
새벽빛, 돌이 굴려진 무덤, 빈 세마포, 놀란 목격자들의 실루엣, 차갑고 푸른 새벽과 따뜻한 금빛의 대비
```
