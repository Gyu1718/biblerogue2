# BibleRogue2 공개 안내

BibleRogue2는 성경 기반 비상업적 텍스트 선택형 서사 게임 프로토타입입니다.

## 비상업 프로젝트

이 프로젝트는 현재 상업적 판매, 유료 서비스, 광고 수익화를 목적으로 하지 않습니다. 공개 목적은 개인 개발, 테스트, 성경 기반 서사 실험, 인터랙티브 콘텐츠 연구입니다.

## 개발 중인 테스트 버전

현재 공개된 GitHub Pages 버전은 정식 완성본이 아닙니다. 일부 기능, 이미지, 문장, 분기 구조, 모바일 대응, 저장 기능에서 오류가 있을 수 있습니다.

정식 완료 판정은 별도의 QA 과정을 통과한 뒤에만 할 수 있습니다.

## 성경 이야기 사용 원칙

이 프로젝트는 성경 사건 자체를 바꾸는 게임이 아닙니다. 플레이어는 성경의 직접 주인공이 아니라, 사건을 목격하고 반응하는 이름 없는 증인의 위치에 섭니다.

선택의 결과는 성경 사건의 변경이 아니라, 증인의 신뢰, 두려움, 기억, 공동체성, 분별, 지체, 흩어짐 같은 태도의 방향으로 표현됩니다.

## 저작권 및 사용 제한

본 프로젝트의 코드, 문장, 이미지, 기획 문서, 게임 구조, 에셋은 별도 허락 없이 상업적으로 사용할 수 없습니다.

현재 별도의 오픈소스 라이선스를 부여하지 않았으므로, 명시적 허락 없이 복제, 재배포, 수정 배포, 상업적 이용을 허용하지 않습니다.

개인적인 감상, 테스트, 링크 공유는 가능합니다.

## SNS 공유용 문구

BibleRogue2는 성경 사건을 바꾸는 게임이 아니라, 성경 이야기 앞에 선 이름 없는 증인의 선택을 따라가 보는 비상업적 텍스트 서사 게임 프로토타입입니다. 현재 개발 중인 테스트 버전입니다.
