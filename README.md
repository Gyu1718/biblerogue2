# BibleRogue2

BibleRogue2는 성경 이야기를 기반으로 한 비상업적 텍스트 선택형 서사 게임 프로토타입입니다.

## 프로젝트 소개

이 프로젝트는 성경 사건 자체를 바꾸는 게임이 아닙니다. 플레이어는 모세, 여호수아, 다윗, 예언자, 사도와 같은 성경의 직접 주인공이 아니라, 그 사건을 목격하고 반응하는 이름 없는 증인의 위치에 섭니다.

선택지는 성경 사건의 결과를 바꾸기 위한 장치가 아니라, 사건 앞에서 드러나는 신뢰, 두려움, 기억, 공동체성, 분별, 지체, 흩어짐 같은 내적 방향을 표현하기 위한 장치입니다.

## 현재 상태

현재 공개된 버전은 정식 완성본이 아니라 개발 중인 테스트 버전입니다.

- Part I. 1장 「출애굽」: 플레이 가능
- Part I. 2장 「광야」: 플레이 가능
- Part I. 3장 「여리고」: 테스트 플레이 가능, 최종 QA 대상
- 나머지 챕터: 준비 중

화면 구성, 문장, 분기 구조, 이미지 연결, 모바일 대응은 계속 수정될 수 있습니다.

## 공개 목적

이 프로젝트는 상업적 판매나 수익화를 목적으로 하지 않습니다. 현재 공개 목적은 다음과 같습니다.

- 성경 기반 선택형 서사 구조 실험
- 청소년과 성인을 위한 신앙적·문학적 인터랙티브 콘텐츠 가능성 탐색
- 텍스트 게임 형식을 통한 성경 이야기 이해 방식 연구
- 개인 개발 및 공개 테스트

## 신학적·서사적 원칙

- 성경 사건 자체를 임의로 바꾸지 않습니다.
- 플레이어를 성경의 중심 인물로 설정하지 않습니다.
- 선택의 결과는 역사적 사건 변경이 아니라 증인의 태도와 공동체의 방향으로 표현합니다.
- 배드엔딩은 성경 사건 실패가 아니라, 말씀 거부·공동체 붕괴·우상 숭배·두려움에 사로잡힌 증인의 실패로 이해합니다.

## 저작권 및 사용 안내

본 저장소의 코드, 문장, 이미지, 기획 문서는 별도 허락 없이 상업적으로 사용할 수 없습니다.

현재 별도의 오픈소스 라이선스를 부여하지 않았습니다. 따라서 명시적 허락 없이 본 프로젝트의 코드, 이미지, 문장, 에셋을 복제·재배포·상업적 용도로 사용하는 것을 허용하지 않습니다.

개인적인 감상, 테스트, 링크 공유는 가능합니다.

## 안내 문구

공유 시에는 다음과 같이 소개할 수 있습니다.

> BibleRogue2는 성경 사건을 바꾸는 게임이 아니라, 성경 이야기 앞에 선 이름 없는 증인의 선택을 따라가 보는 비상업적 텍스트 서사 게임 프로토타입입니다. 현재 개발 중인 테스트 버전입니다.

## 개발 상태 관련 안내

개발 중인 프로젝트이므로 일부 화면, 이미지, 문장, 분기, 저장 기능에서 오류가 있을 수 있습니다. 정식 배포 전에는 별도의 QA 과정을 거쳐야 합니다.
